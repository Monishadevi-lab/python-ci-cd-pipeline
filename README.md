# Python CI/CD Pipeline Project

This project demonstrates a simple CI pipeline using GitHub Actions.

## Technologies
- Python
- GitHub Actions

## Workflow
The CI pipeline runs automatically whenever code is pushed to the repository.

Steps performed:
1. Checkout repository
2. Install Python
3. Install dependencies
4. Run the Python application

## Output
The pipeline executes the Python script and prints a message to verify successful execution.

This project demonstrates basic Continuous Integration (CI) concepts used in DevOps workflows.
